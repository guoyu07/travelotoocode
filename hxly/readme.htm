 《思途CMS网站管理系统》安装说明

一. 运行环境需求: 
PHP5.2、5.3、5.4，php.ini中register_globals 设置为 Off，并需要开启php扩展：php_curl、php_mbstring、php_gd2、php_mysql、php_zip
MYSQL5.0以上，需要开启MYIASM和InnoDB引擎支持
伪静态功能的开启：IIS系列服务器，请安装isapi_rewrite3扩展（下载组件）（配置教程）
apache系列服务器，请开启伪静态支持（配置教程）

二. 安装步骤:
第 1 步：上传
将安装包目录中的全部文件和目录结构上传到服务器的网站根目录（注意是上传目录中的所有文件和目录）。

其他说明：要安装在80端口，否则不能正常升级和正版验证。

第 2 步：设置目录属性
设置目录权限可以使用以下方法：
  • 你可以使用ftp软件将所有目录及其子目录、文件属性设置为777。

  • linux平台下你有shell权限则通过执行chmod来修改目录权限。

  • windows平台下需要给目录分配IIS帐号读写权限。
    		•

    第 3 步：执行安装脚本
  • 在执行网站之前，我们需要给当前网站的默认首页设置成index.php。

  • 在浏览器中访问你的域名（网址）,如果默认首页index.php设置成功，域名（网址）会跳转到
     http://你的域名（网址）/install/index.php

  • 安装脚本会检查您的服务器系统环境、数据库环境，并具备一定的纠错功能。如果您在之前某一步骤操作有问题，通常安装脚本会发现并作以提示，请您根据提示再对安装过程进行检查。如果没有提示出问题，请您按照其中的说明，完成最后的安装。
    		•

    填写参数时注意以下几点：
  • 若不清楚mysql服务器地址，用户名，密码，数据库名可向虚拟空间提供商咨询。

  • 填写管理员信息时设置一个安全且不易忘记的密码。
    		•


    第 4 步：初始化设置
    安装完成后，在浏览器中访问 http://你的域名(网址)/newtravel/ 页面，输入正确的用户名和密码就可登陆后台管理。


    三. 至此，思途CMS网站管理系统全部安装完毕。
